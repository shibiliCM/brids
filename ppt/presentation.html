MODEL PIPELINE

Dynamic Checkpoint Loading

During training, the model saved standard ResNet-18 with 1,000 logits instead of truncating to 200, which silently broke the custom model loader.

Our custom loader inspects the saved state dictionary dynamically, sets up the linear layer dimension to exactly match the checkpoint shape (`1000`), and runs strict state dict loading to avoid discarding trained weights.

CODE IMPLEMENTATION

The Predict Function

The core inference function processes input images, runs forward propagation, and handles logit alignment.

  • 🔒 No Gradients: Wrapped in a torch.no_grad() block to optimize speed and RAM.
  • ✂️ Logit Slicing: Extracts the first 200 elements (output[:, :200]) to dynamically handle checkpoints saved with 1000 outputs.
  • ⚡ Top-K Sorting: Calculates Softmax, and runs torch.topk to select Top-5 index/probability pairs.
python def predict(model, img_tensor, confidence_threshold=0.5): with torch.no_grad(): output = model(img_tensor) # Slice output to matching 200 CUB classes logits = output[:, :200] probs = torch.softmax(logits, dim=1).squeeze(0) if probs.dim() == 0: probs = probs.unsqueeze(0) # Extract Top-5 predictions k = min(5, len(probs)) topk = torch.topk(probs, k) top_idx = topk.indices.tolist() top_probs = topk.values.tolist() return list(zip(top_idx, top_probs))

EXPLAINABLE AI

🔥 Show Grad-CAM Heatmap

Grad-CAM (Gradient-weighted Class Activation Mapping) makes black-box deep learning models transparent by illustrating visual attention.

  • 🎯 Target Layer: Attaches hooks to ResNet-18's last convolutional layer (layer4[-1].conv2).
  • 🧮 Gradient Pooling: Computes the gradient of the target class score with respect to feature maps, pool-averaging channel weights.
  • 🌈 Heatmap Overlay: Blends jet-colormap values with the original image to showcase the exact regions driving the classification.
python def generate_heatmap(self, input_tensor, class_idx): self.model.eval() self.model.zero_grad() output = self.model(input_tensor) score = output[0, class_idx] score.backward() weights = self._gradients.mean(dim=[0, 2, 3]) acts = self._activations[0] cam = (weights[:, None, None] * acts).sum(dim=0) cam = torch.relu(cam).numpy() return (cam - cam.min()) / (cam.max() + 1e-8)

HYBRID RECOGNITION

🔍 Enable CLIP Fallback

Handles Out-of-Distribution (OOD) images using zero-shot learning to avoid misleading predictions.

  • 🛡️ OOD Protection: Activates when the ResNet-18 top-1 confidence falls below the configurable threshold (default: 70%).
  • 🧬 OpenCLIP ViT-B-32: Utilizes OpenAI's pretrained vision-language model to match the image features against text prompts.
  • 📋 Candidate List: Zero-shot parses the image against a list of common birds not found in CUB-200 (Crows, Pigeons, Parrots, Owls, etc.).
TRADE-OFF ANALYSIS

System Pros & Cons

🟢 Pros & Advantages

  • ✅ Enhanced Trust: Grad-CAM highlights key features (e.g. wings, beak) showing how predictions are reached.
  • ✅ Graceful Fallbacks: CLIP prevents embarrassing failures by correctly suggesting common species not in the CUB dataset.
  • ✅ Configurability: Real-time slider adjusting OOD threshold enables custom balance of accuracy vs. permissiveness.

🔴 Cons & Limitations

  • ❌ Increased Footprint: Adding OpenCLIP model adds ~300MB RAM/VRAM load on first run initialization.
  • ❌ Inference Latency: Dual-model architecture adds small latency when OOD activates and triggers fallback inference.
  • ❌ MSP Heuristics: Maximum Softmax Probability is a basic baseline; can sometimes yield false confidence in OOD images.
FAQ SECTION

Questions & Answers

Q: Why is ResNet-18 chosen over larger models like ViT or ConvNeXt?

ResNet-18 provides the optimal balance between accuracy and computational cost for local deployment on standard CPU containers, while still supporting real-time Grad-CAM calculation without delay.

Q: How does Maximum Softmax Probability (MSP) detect out-of-distribution birds?

A classifier trained only on CUB-200 species spreads its probability density broadly across classes when fed an unknown bird. Since no single class scores highly, the maximum softmax probability remains low, triggering OOD detection.

Q: Can the OOD threshold slider be auto-calibrated?

Yes. Using temperature scaling (calibration) on a held-out validation set of combined CUB and non-CUB images, we can mathematically calculate the threshold that maximizes the F1-score of OOD classification.
